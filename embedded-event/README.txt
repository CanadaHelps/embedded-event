Adds a shortcode to support embedded event pages. [embedevent id="168"]

== Description ==
Allow your supporters to buy tickets for your event without leaving your site.

== Installation ==

1. Install and activate the plugin on the Plugins page
2. Add shortcode to page or post content (ex. `[embedevent id="168"]`)

== Other Notes ==

= embedevent params: =
* **id** - a CanadaHelps Event page id
* **language** - [**fr** or **en**] desired language (default: **en**)

== Frequently Asked Questions ==
= How do I display my event in French? =
Add **language** parameter to the shortcode.
`[embedevent id="168" language="fr"]`
